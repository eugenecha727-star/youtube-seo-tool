🎬 YouTube SEO Tool

Редактируй заголовки и описания прямо здесь — изменения применяются на твоём канале через YouTube API

⚙️ Шаг 1 — Настройка Google OAuth
1
Зайди на console.cloud.google.com → создай проект "YouTube SEO Tool"
2
APIs & Services → Library → найди и включи YouTube Data API v3
3
Credentials → Create Credentials → OAuth 2.0 Client ID → тип Web application
4
В поле Authorized JavaScript origins добавь домен с которого открываешь этот файл (например http://localhost или адрес сайта)
5
В поле Authorized redirect URIs добавь тот же адрес
6
OAuth consent screen → добавь свой email как Test User
🔑 Шаг 2 — Введи Client ID
Где найти Client ID:
  • Google Cloud Console → Credentials → твой OAuth 2.0 Client ID → скопируй "Client ID"
